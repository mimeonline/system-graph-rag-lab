# ADR-0006 Umsetzungsfokus E1 Local Profil bei unverändertem Public Zielbild

## Status
1. Accepted am 2026-02-25.

## Kontext
1. Die Architektur trennt bereits die Laufzeitprofile `local` und `public`.
2. Für die aktuelle Umsetzung wird eine eindeutige Reihenfolge benötigt, damit Dev und QA deterministisch arbeiten können.
3. PM plant die story-technische Trennung im nächsten Run, Architect muss dafür eine belastbare Entscheidungsgrundlage liefern.
4. Public Runtime auf Vercel plus Neo4j Aura bleibt das verbindliche MVP Zielbild.

## Entscheidung
1. Aktueller Umsetzungsfokus in E1 liegt auf dem Profil `local`.
2. E1 umfasst Next.js lokal und Neo4j Docker als primären Betriebsweg für Implementierung und QA Basischecks.
3. Das Profil `public` bleibt unverändert als Zielbild für spätere E4 Stories.
4. Bis E4 sind nur lokale Betriebsadapter zu implementieren, solange API und Retrieval Contract identisch zum Public Zielbild bleiben.
5. Eine Verschiebung dieses Fokus benötigt eine neue ADR oder ein explizites Architect Update.

## Begründung
1. Local First reduziert externe Abhängigkeiten in der frühen Umsetzungsphase.
2. Gleichbleibende Contracts verhindern spätere Migrationsbrüche zwischen E1 und E4.
3. Dev und QA erhalten eine klare Reihenfolge ohne Scope Erweiterung.
4. Das Public Betriebsziel bleibt verbindlich dokumentiert und wird nicht aufgeweicht.

## Konsequenzen
1. Dev priorisiert lokale Startfähigkeit, lokale Neo4j Erreichbarkeit und lokale Rate Limit Adapter.
2. QA priorisiert Contract Tests im lokalen Profil mit denselben Akzeptanzkriterien wie public.
3. Public Infrastrukturtests werden nicht vorgezogen, sondern in E4 fokussiert.
4. Architektur und Spezifikationen müssen die Profiltrennung und die Reihenfolge explizit benennen.

## Verworfene Alternativen
1. Gleichzeitige Umsetzung von local und public wurde verworfen, da sie E1 unnötig verlangsamt.
2. Public First wurde verworfen, da es frühe Iterationen von externen Betriebsdiensten abhängig macht.
3. Lokales Profil mit abweichendem API Contract wurde verworfen, da es spätere Integrationsrisiken erhöht.
