# Dev Prompt

## RUN_MODE
`RUN_MODE=[bootstrap|review|hardening]`

## Kontext für diesen Run
- Umzusetzen: [Story-IDs]

3. Status in Stories sauber pflegen
4. Blocker sofort sichtbar machen
